1	0	4
2	0	3
6	2	0
3	0	2
4	0	1
5	0	0
기본값으로 부모글(1~5)을 0으로, 순서는 입력시마다 1씩 증가를 하면
답글(6)을 입력시 순서는 위와 같음.

글번호	부모글	순서
1	0	5
2	0	4
6	2	3
3	0	2
4	0	1
5	0	0
이 되기를 원하므로 방법은 아래와 같음

답글(6)을 작성하는 댓글(2)의 순서(3)를 가져와서
답글(6)의 순서(3)로 지정(setter로 값만지정하고 insert는 하지 않음)
그리고 가져온 댓글(2)의 순서(3)와 같거나 큰수를 +1하고
답글을 삽입하면 그 사이로 들어감
